# netty+websocket实现在线聊天

| 属性值 | 属性常量   | 描述                                          |
| ------ | ---------- | --------------------------------------------- |
| 0      | CONNECTING | 正在与服务端建立WebSocket连接，还没有连接成功 |
| 1      | OPEN       | 连接成功并打开，可以发送消息                  |
| 2      | CLOSING    | 进行关闭连接的操作，且尚未关闭                |
| 3      | CLOSE      | 连接已关闭或不能打开                          |
